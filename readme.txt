== Description ==

= Image to text. =

* Can specify the font size.
* Can specify the background color.
* Can specify the font color.
* Can specify the font.
* Can specify the alt.
* The [fonts "Noto Sans JP"](https://fonts.google.com/earlyaccess#Noto+Sans+JP) are bundled.

== Installation ==

1. Upload `image-to-text` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
